There are 3 types of lists:

definition lists
which show terms and definition
unordered lists
which show bullets
ordered list
which show number of letters

Above is an example of a definition list

Unordered lists can have a type attribute which can be applied to the ul or the li tag. The three values allowed are:

  • type="square"
  • type="disc"
  • type="circle"

Ordered lists can have a type attribute too. Used along with a start attribute they can show

  1. lower case letters
  1. upper case letters
  1. lower case roman numerals
  1. upper case roman numerals
  1. numbers
